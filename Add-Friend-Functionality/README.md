# Card App

This is a simple card app that allows you to add or remove a friend by clicking a button.

## Overview

This app consists of an HTML file (`index.html`), a CSS file (`style.css`), and a JavaScript file (`script.js`). It displays a card with a person's name, status, and an image. You can add or remove the person as a friend by clicking the "Add Friend" button.

## Usage

To use this app, simply open the `index.html` file in a web browser. Click the "Add Friend" button to change the status to "Friends" and the button text to "Remove Friend". Click the button again to revert the changes.

## File Structure

- `index.html`: Contains the HTML structure of the card app.
- `style.css`: Defines the styles for the card app.
- `script.js`: Contains the JavaScript logic to toggle the friend status.
