# Password Generator

This is a simple password generator project created using HTML, CSS, and JavaScript.

## Description

This project generates random passwords based on user-defined criteria such as length and character types (uppercase letters, lowercase letters, numbers, and special characters). It provides a user-friendly interface to customize and generate passwords securely.

## Live Preview

You can view the live preview of the password generator [here](https://invtfl-bharti.github.io/password-generator/).

## Features

- Customizable password length
- Option to include/exclude uppercase letters, lowercase letters, numbers, and special characters
- Instant password generation with a single click
- Copy password to clipboard functionality
- Color-coded strength indicator based on password length

## Usage

1. Open the [live preview](https://invtfl-bharti.github.io/password-generator/) in your web browser.
2. Customize your password criteria by adjusting the slider and checkboxes.
3. Click the "Generate Password" button to generate a random password.
4. Optionally, click the copy icon to copy the generated password to your clipboard.
5. Enjoy your secure and randomly generated password!
